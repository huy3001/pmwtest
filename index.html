Digital goods

Digital Goods API is mainly focused on selling In-game virtual items, services and premium membership with PayAlto.

Stored widget call

It shows a widget, where stored products in Products section of Merchant Account are available for payment. Both subscription and one-time payment are supported in our products section.

Parameter:

Name Required Format Description
key yes String(32) Sample:
048c51f8fc834a1467db96b683f16e70 Can be found in General Settings of the Project inside of your Merchant Account
uid yes String(64) Sample: test@paymentwall.com
Not case-sensitive
ID of the end-user in your system (usually it is an email) who is viewing the widget. The maximum length is 64.
widget yes String Sample: p1_1
Widget code. Can be obtained by Merchant Account -> Projects -> Widgets
email yes String Sample: test@paymentwall.com
Paymentwall will automatically send a payment receipt to the user.
timestamp yes String(10) Sample: 1477985778
Unix timestamp of the current date
sign_version no String Sample: 2, 3
Signature Version.
sign no String Sample: 7633c3b060fa2ddd41283ffd4629ab40
Refer to Signature Calculation.

Result:

You will get a URL or iframe by widget object.
